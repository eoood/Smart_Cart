# Smart_Cart
스마트카트로 앱 결제 기능과 매장관리 시스템



##[팀 프로젝트] 스마트카트

[ 시연영상 ]



##설명
앱을 통해 당일 학생과 교직원 식당의 메뉴 정보를 알 수 있다.
원하는 메뉴를 선택 후 장바구니에 담아놓거나 결제할 수 있다.

##지원
Git
Android Emulator

#3사용 기술
Java
SQLite

##개발 환경
Window OS
Android Studio 2.3.3

##개인 역할
안드로이드 내장 데이터베이스인 SQLite를 이용한 로그인, 회원가입, 아이디/비밀번호 찾기 구현.
장바구니, 고객문의 페이지 구현.
